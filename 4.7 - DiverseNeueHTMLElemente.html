Navigation

Damit wir auf einer Seite per Klick direkt zu den gewünschten Abschnitten navigieren können, müssen wir die Abschnitte bearbeiten und zudem eine Navigationsliste erstellen.

Die folgende Liste ist in einem <nav> Element und darin befinden sich a-Tags. Die Überschriften müssen mit einer id versehen werden. So hat die Überschift "Tabellen" die id=tabellen. Das ist nichts anderes als eine class, also eine Benennung.

  • Tabellen
  • Videos
  • Input Boxen
  • Dropdown Menü

Tabellen

Tabellen benötigen viel CSS und "gut" auszusehen. Die Attribute dafür sind aber zum Teil bereits bekannt, denn width height padding font-size usw. können hier benutzt werden. Ich kann im CSS sowohl die ganze Tabelle (table{...}) ansteuern oder auch nur die einzelnen Zeilen oder Spalten (th{...} oder td{...})
